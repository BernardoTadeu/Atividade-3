  • HTML
Conceito=
HTML é a base/estrutura de um arquivo WEB, sendo utilizado para definir suas características básicas (como: texto; imagem; tabelas; organização básica...).


Estrutura inicial=
< !DOCTYPE html >
< html >
< head >
< meta charset="utf-8" >
< title > A sua escolha < /title >
< / head >
< body >
Aqui o HTML é desenvolvido.
< / body >
< /html >

Você também pode digitar "html:5" e dar ENTER.


Códigos básicos=
< br > = Pular uma linha

< fieldset > TUDO QUE DEVE FICAR DENTRO DO BLOCO DEVE SER FEITO AQUI < /fieldset > = Criar um bloco de divisão

< div > TUDO NESSA "DIVISÃO" DEVE FICAR AQUI< /div > = Divide elemento do HTML e facilita a sua identificação

"required" no final do < input > = Torna tal resposta obrigatória.


Adicionando imagens=

Adicione imagens (que devem estar na pasta que o HTML está salvo) inserindo este código:

< img scr="NOME DA IMAGEM" width="DEFINE A LARGURA" heigth="DEFINE A ALTURA"> Somente imagem <




< img scr="NOME DA IMAGEM" alt="O QUÊ SERÁ LIDO PELO PROGRAMA PARA CEGOS" > Imagem + Legenda para cegos <




< figure >
< img scr="NOME DA IMAGEM" alt="O QUÊ SERÁ LIDO PELO PROGRAMA PARA CEGOS" >
< figcaption >
LEGENDA NORMAL
< / figcaption >
< / figure >

Imagem + Legenda para cegos + Legenda normal /\



Criando listas=
Para criar uma lista simples, insira este comando:
< ol >
< li > 1° ELEMENTO DA LISTA < /li >
< li > 2° ELEMENTO DA LISTA < /li >
< li > 3° ELEMENTO DA LISTA < /li >
< li > ... < /li >
< /ol >

  • 1° ELEMENTO DA LISTA
  • 2° ELEMENTO DA LISTA
  • 3° ELEMENTO DA LISTA
  • ...



    • Para criar uma lista numerada, insira este comando:

    < ul >
    < li > 1° ELEMENTO DA LISTA < /li >
    < li > 2° ELEMENTO DA LISTA < /li >
    < li > 3° ELEMENTO DA LISTA < /li >
    < li > ... < /li >
    < /ul >

    • 1° ELEMENTO DA LISTA
    • 2° ELEMENTO DA LISTA
    • 3° ELEMENTO DA LISTA
    • ...



    Em relação ao "< ol >" é possível fazer diferentes tipos de "Listas Simples", como:

    < ol type="a" > = Lista na ordem alfabética (letras minúsculas)
    < ol type="A" > = Lista na ordem alfabética (letras maiúsculas)
    < ol type="i" > = Lista em "i", como: i, ii, iii
    < ol type="I" > = Lista em "I", como: I, II, III


    Em relação ao "< ul >" é possível fazer diferentes tipos de "Listas Numeradas", como:
    • < ul type="square" > = Lista com o símbolo de um quadrado
    • < ul type="circle" > = Lista com o símbolo de um círculo

    Fazendo Formulários=
    Para criar um pequeno espaço de resposta, para uma curta pergunta (normalmente), insira este código:

    < label > ESCREVA SUA PERGUNTA AQUI < /label >
    < input type="text" id="DIGITE UM NOME, PARA IDENTIFICAR ESSA PERGUNTA DEPOIS" >




    Para perguntar o email de alguém, insira este código:

    < label > SOLICITE SEU EMAIL < /label >
    < input type="email" id="DIGITE UM NOME, PARA IDENTIFICAR ESSA PERGUNTA DEPOIS" >




    Para que a questão seja de múltipla escolha (no formato de marcar círculos), insira este código:

    < label > ESCREVA SUA PERGUNTA < /label >
    < br > < br >
    < input type="radio" name="DIGITE O NOME DESSA OPÇÃO AQUI" > ESCREVA A RESPOSTA DESSA OPÇÃO
    < br >
    < input type="radio" name="DIGITE O NOME DESSA OPÇÃO AQUI" > ESCREVA A RESPOSTA DESSA OPÇÃO
    < br >
    < input type="radio" name="DIGITE O NOME DESSA OPÇÃO AQUI" > ESCREVA A RESPOSTA DESSA OPÇÃO
    < br >
    < input type="radio" name="DIGITE O NOME DESSA OPÇÃO AQUI" > ...
    < br >

    Para que seja possível marcar apenas uma opção é necessário que todos os "inputs" tenham o mesmo nome.
    Para que seja possível marcar mais de uma opção é necessário que os "inputs" tenham nomes diferentes.

    < textarea maxlength=" DEFINE O NÚMERO MÁXIMO DE PALAVRAS" style="width: 28em height: 28em" > < /textarea > = Com o "style", o tamanho do box da área de texto é definido.




    Para criar o botão de enviar (mas que não vai ter funcionalidade ainda), insira este código:

    • CSS

    Conceito:

    CSS é a parte do arquivo WEB que define suas características visuais, assim definindo elementos estéticos do site



    Para vincular um arquivo CSS a um HTML, inisra este código no < head >:

    < link rel="stylesheet" type="text/css" href="NOME DO ARQUIVO CSS" >



    Conhecimentos básicos:

    Usa-se " . " antes de CLASSes e " # " antes de ID.

    O símbolo " * " é utilizado para se referir a toda página WEB.

    Para fazer anotações que não apareceram no texto, usa-se "/* */".



    Para definir a cor de determinado texto, insira este código (isso no arquivo de CSS):

    #ESCREVA O ID DO "BLOCO/DIVISÃO" QUE O TEXTO ESTÁ INSERIDO {
    color: ESCREVA O NOME DA COR DESEJADA;
    }



    Para definir a cor do fundo de certa divisão ou de todo site, insira este código:

    #ESCREVA O ID DO "BLOCO/DIVISÃO" QUE O FUNDO SERÁ ALTERADO {
    background-color: ESCREVA O NOME DA COR DESEJADA;
    }

    Para criar esse efeito de transição da cor, insira este código no CSS:

    #ID DO FIELDSET{ background-image: linear-gradient(to bottom, #37fff5, #0ebbff); }



    Para alterar a ingremidade das pontas de uma divisão/fieldset, insira esta código:

    #ESCREVA O ID DO "FIELDSET/DIVISÃO" A SER ALTERADO {
    border-radius: DIGITE O NÚMERO DE PIXELS QUE A INCLINAÇÃO TERÁ (COMO 15 px);
    }



    Para alterar o tamanho da borda de um bloco/fieldset, insira este código:

    #ESCREVA O ID DO "FIELDSET/DIVISÃO" A SER ALTERADO {
    border-width: TAMANHO DA BORDA (como 15px);
    }



    Para alterar a fonte do texto, insira este código:

    .ESCREVA O ID DO "BLOCO/DIVISÃO" EM QUE O TEXTO ESTÁ INSERIDO{
    font-family: ESCREVA O NOME DA FONTE (Georgia, por exemplo);
    }



    Para alterar o tamanho do texto, insira este código:

    .ESCREVA O ID DO "BLOCO/DIVISÃO" EM QUE O TEXTO ESTÁ INSERIDO{
    Font-size: TAMANHO DA LETRA (como: 10px;);
    }



    Para fazer com que ao carregar o site, os elementos apareçam lentamente, insira este código (TODOS OS ELEMENTOS COM ESSE EFEITO DEVEM TER A class="fadeIn"):

    .fadeIn {
    animation: fadeIn 2s (por exemplo) linear
    }
    @keyframe fadeIn {
    from { opacity 0;}
    to {opacity 1;}
    };



    Para alterar características de um botão (por exemplo, mas também serve em outros, como imagens), insira este código:

    button (SE FOR UM BOTÃO) {
    margin-left:570px;
    padding: 8px;
    color:black;
    background-color: yellow;
    }
    button:hover{
    margin-left:555px;
    padding: 20px;
    color:white;
    background-color: rgb(0, 4, 255);
    border-radius: 18px;
    transition: 1s;
    }
